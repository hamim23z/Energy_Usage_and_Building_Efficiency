# Energy_Usage_and_Building_Efficiency

**TO RUN INTERACTIVE WEBPAGE**
you need to have flask and DuckDB(python library) installed
either conda/pip install them
go to top level and run "python app.py" in terminal

**CODE FOR PRESENTATION**
info.ipynb contains all code used to make plotly graphs in jupyter slides
The actual slides is in the templates folder, "info.slides.html". It can be opened alone offline or
the accessed through running the webpage.

**REPORT**
report can be found via hyperlink in webpage and in pdf version in repo

**DATA**
the data can be found at 
https://www.kaggle.com/datasets/ashioyajotham/nyc-geojson-file
https://data.cityofnewyork.us/Environment/NYC-Building-Energy-and-Water-Data-Disclosure-for-/5zyy-y8am/about_data
run them by the parser if DB is missing Data (it shouldnt be)
